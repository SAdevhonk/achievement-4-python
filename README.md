# Achievement 4 – Python Data Analysis Project

## Project Overview
This project was completed as part of CareerFoundry’s Data Analytics Program (Achievement 4).  
The objective of the project was to complete a structured data analysis using Python, from data preparation through analysis and reporting.

The project follows best practices for coding etiquette, documentation, and project organization.

## Tools Used
- Python
- Jupyter Notebook
- Excel
- Microsoft Word

## Key Skills Demonstrated
- Data cleaning and preparation
- Exploratory data analysis
- Writing clear and well-documented Python code
- Organizing analysis using a structured folder system
- Communicating insights through notebooks and reports

## Repository Contents
This repository mirrors the folder structure finalized in Exercise 4.10 and includes:
- Jupyter notebooks containing the analysis
- Supporting files such as reports and documentation

Note: Raw datasets are intentionally excluded from this repository.

## Additional Notes
This repository is intended to showcase analytical thinking, code quality, and project organization rather than raw data.
